プライバシーポリシー（PatchLOL）

最終更新日: 2026年3月19日

本ポリシーは、Discord ボット「PatchLOL」（以下「本ボット」）の利用にあたり、どのような情報が関係するかを説明するものです。本ボットはオープンソースの自己ホスト型です。実際にデータを処理するのは、ボットを起動している当事者（サーバー管理者・運用者）です。

1. 本ボットが扱う情報（運用者の環境）

運用者が本ボットを動かす際、次の情報が運用者のマシンまたはホストに保存されることがあります。

  • Discord ボットトークン（環境変数または .env）— Discord API への認証にのみ使用します。
  • サーバー ID とチャンネル ID（patch_guild_channels.json 等）— /patch set で指定した通知先を記録します。
  • パッチ通知の状態（patch_state.json）— 最後に通知した LoL/TFT のパッチ情報・送信済みチャンネル ID などを重複送信防止のため保存します。

これらはDiscord 社のサーバー上に本ボット作者が収集・保管するものではありません。運用者の責任で管理してください。

2. Discord 経由で送信される情報

  • 本ボットは、設定されたテキスト／お知らせチャンネルに、Riot Games 公式サイトから取得したパッチノートに関するメッセージ（埋め込み・画像等）を投稿します。
  • スラッシュコマンド /patch 利用時、Discord が定める範囲でコマンド実行者・サーバー情報がインタラクションとして本ボットに渡ります（権限確認等）。

3. 第三者サービス

  • Discord — 利用規約・プライバシーは Discord プライバシーポリシー に従います。
  • Riot Games（leagueoflegends.com 等） — パッチ一覧・パッチノートページの取得に利用します。取得は公開 Web ページへの HTTP リクエストです。

4. お問い合わせ

本ポリシーはリポジトリ管理者が定めます。連絡先は、本ボットを配布している GitHub リポジトリの Issue 等をご利用ください（運用者ごとに異なります）。
